# HR-Sensor w/ NeoPixel

## Hardware Assembly

Sensor:               Arduino: <br/>
PulseSensor (signal)  A0	  Analog input
PulseSensor (VCC)	    5V	  Power
PulseSensor (GND)	    GND	  Ground
NeoPixel Data In	    6	    Digital output pin
NeoPixel VCC	        5V	  Power
NeoPixel GND	        GND	  Ground
